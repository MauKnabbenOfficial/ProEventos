npm --version (versao rodando npm)
nvm list (lista versoes npm instaldas)
node -v (versao rodando node)
ng --version (versao rodando angular)
ng new ProEventos-App (cria um novo projeto angular)
npm start (starta projeto angular)
ng g n <nome> <caminho> (gera novo componente [caminho default: dentro de src/app])

dotnet watch run (compila e executa aplicação dotnet com watch no terminal)

pipe: {{eventos | json}} (formata array para json)

dotnet new sln -n --<nome> (cria uma nova solução no diretorio atual)
dotnet new classlib -n --<nome> (cria uma nova biblioteca classlib no diretorio atual)

dotnet add <destino>.csproj reference <origem>.csproj (para referencia entre projetos)
dotnet ef migrations add Initial -p <Origem> -s <Destino>
